# README

HITLUG 社刊仓库

## Roadmap

- alpha 阶段：经由少数几名社团成员进行直接的内容填充，所有提交直接作用于 hitlug/club-journal 仓库，完善贡献指南
- beta 阶段：为本仓库开启写保护，应用GitHub Flow/GitLab Flow，引入静态网页生成器和CI，完善术语表，完善贡献流程，提供 CLA
- gamma 阶段：使用定制的半自动化设施进行多格式来源的稿件收录、合并、修订、发版
