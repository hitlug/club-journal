# README

HITLUG 社刊仓库

---

本仓库系 *哈尔滨工业大学 Linux 开源学生社团* 社刊仓库，用于社刊的投稿、维护、发版等操作。

## Contributing

我们接收任何有关于如下内容的投稿：

- 技术资讯与新闻
- 学术资讯、论文品读
- 计算机知识分享与经验技巧
- （一校三区）校园、社团相关的资讯、新闻与摄影作品

投稿内容在主题上应与本社团相合，我们欢迎计算机技术、软件工程、网络安全、开源技术栈、Linux 相关内容等主题的投稿。

详情请参阅 [CONTRIBUTING.md](./CONTRIBUTING.md) 文件。

## Roadmap

- alpha 阶段：经由少数几名社团成员进行直接的内容填充，所有提交直接作用于 hitlug/club-journal 仓库，完善贡献指南
- beta 阶段：为本仓库开启写保护，应用GitHub Flow/GitLab Flow，引入静态网页生成器和CI，完善术语表，完善贡献流程，提供 CLA
- gamma 阶段：使用定制的半自动化设施进行多格式来源的稿件收录、合并、修订、发版

## Contact

社刊内容上的问题请尽量通过 GitHub Issue 与我们沟通。

其他联系方式，请翻阅社团网站上的[联系方式](https://hitlug.org/about.html#联系方式)页面。
