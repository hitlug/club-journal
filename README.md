# README

HITLUG 文章归档仓库

---

本仓库系 *哈尔滨工业大学 Linux 开源学生社团* 文章归档仓库，用于社团文章的投稿、维护、发版等操作。

## Contributing

本归档主要面向校内对Linux与开源技术感兴趣的师生，内容涵盖但不限于：

- 技术资讯与新闻
- 学术资讯、论文品读
- 计算机知识分享与经验技巧
  - 技术经验分享
  - 开源工具实践
  - 计算机技术的跨学科应用
- （一校三区）校园、社团相关的资讯、新闻与摄影作品
- 其他与本校、本社团、开源技术相关的文章、故事与艺术作品

归档内容在主题上应与本社团相合，我们欢迎计算机技术、软件工程、网络安全、开源技术栈、Linux 相关内容等主题的文章归入本归档。

本归档托管于我社团在 GitHub 上的仓库。采取「对内发布内容，对外公开源码，滚动更新」的模式进行维护。

详情请参阅 [CONTRIBUTING.md](./CONTRIBUTING.md) 文件。

## Roadmap

- alpha 阶段：经由少数几名社团成员进行直接的内容填充，所有提交直接作用于 hitlug/club-journal 仓库，完善贡献指南
- beta 阶段：为本仓库开启写保护，应用GitHub Flow/GitLab Flow，引入静态网页生成器和CI，完善术语表，完善贡献流程，提供 CLA
- gamma 阶段：使用定制的半自动化设施进行多格式来源的稿件收录、合并、修订、发版

## Contact

内容上的问题请尽量通过 GitHub Issue 与我们沟通。

其他联系方式，请翻阅社团网站上的[联系方式](https://hitlug.org/about.html#联系方式)页面。
