# 贡献指南

## 项目现状

项目目前处于 _alpha_ 阶段。

## 添加新文章

所有新章节需要声明在 `outline.typ` 中。`outline.typ` 需要包含所有需要被排入主文件的，在 front、body、back 三个子目录下的稿件。

body 下的日期遵循东八区。根据稿件定稿和稿件接受时间进行划分。

所有post文件以如下内容开头：

```typst
#import "../../snippet/funct.typ":*

#newpost(author: "你的GitHub ID", title:"你的文章标题",artitype: "文章类型")[
    // 你的文章内容
]
```

其中 `artitype` 中可以填入如下几种文章类型：

- `"post"`： 对应`"post"`，即一般意义上的文章，可以特指技术领域的文章
- `"news/tech"`： 对应`"技术资讯"`，即技术领域的资讯新闻
- `"news/club"`: 对应`"社团资讯"`，即本社团相关的资讯新闻
- `"news/campus`": 对应`"校园资讯"`，即校园生活相关的资讯新闻
- `"brief"`： 对应`"简讯"`，校园生活、社团、技术等领域的简讯
- `"paper"`： 对应`"论文分享"`，即对技术领域相关论文的分享，包括论文的介绍、拆读、分析、学习笔记等
- `"misc"`： 对应`"杂项"`，即不便于分类，不属于以上任何一类的文章
